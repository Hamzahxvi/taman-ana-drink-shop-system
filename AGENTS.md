# AGENTS.md — Taman Ana Drink Shop System

## Stack

- **Backend**: Laravel 13 + Fortify (auth: registration, passwords, email verification, 2FA)
- **Frontend**: React 19 + Inertia.js 3 + TypeScript + Tailwind CSS 4
- **Testing**: Pest 4 (PHP), Vitest? — no, pest only
- **Tooling**: Vite 8, ESLint 9, Prettier, Pint (PHP), Wayfinder (type-safe routes)

## Developer Commands

```sh
# Dev server (runs artisan serve + queue:listen + Vite concurrently)
composer dev

# Build frontend
npm run build
npm run build:ssr          # with SSR

# Static analysis (order: format → lint → types)
npm run format             # Prettier (resources/)
npm run lint               # ESLint with --fix
npm run types:check        # tsc --noEmit

# PHP lint
composer lint              # pint --parallel
composer lint:check        # pint --parallel --test (dry-run)

# Run all checks locally
composer ci:check          # lint:check + format:check + types:check + test

# Full project setup
composer setup             # composer install → cp .env → key:generate → migrate → npm i → npm build

# Tests
composer test              # config:clear → lint:check → php artisan test (runs Pest)
# Single test file:
./vendor/bin/pest tests/Feature/SomeTest.php
```

## Architecture

| Path | Purpose |
|---|---|
| `routes/web.php` | Web routes (welcome, dashboard) |
| `routes/settings.php` | Auth'd settings routes (profile, security, appearance) |
| `resources/js/app.tsx` | Inertia app entry — auto-resolves layout by page name |
| `resources/js/pages/` | Inertia page components (welcome, dashboard, auth/, settings/) |
| `resources/js/components/` | Shared React components; `ui/` contains Radix UI + shadcn-style primitives |
| `resources/js/layouts/` | AppLayout, AuthLayout, SettingsLayout |
| `resources/js/hooks/` | Custom hooks (appearance, clipboard, mobile, 2FA, etc.) |
| `resources/js/types/` | Shared TS type definitions |
| `resources/js/actions/` | Wayfinder-generated typed helpers (do not edit) |
| `resources/js/wayfinder/` | Wayfinder-generated route types (do not edit) |
| `resources/js/routes/` | Wayfinder-generated per-route modules (do not edit) |
| `app/Actions/Fortify/` | Fortify action classes (CreateNewUser, ResetUserPassword) |
| `app/Http/Controllers/Settings/` | ProfileController, SecurityController |

## Key Conventions

- **Layout resolution** (in `app.tsx`): `welcome` → no layout; `auth/*` → `AuthLayout`; `settings/*` → `AppLayout + SettingsLayout`; everything else → `AppLayout`
- **Import ordering** (enforced by ESLint): builtin → external → internal → parent → sibling → index (ascending, case-insensitive); use `import type` with `separate-type-imports` fix style and top-level type specifiers
- **Brace style** (enforced): 1TB, no single-line allowed; `curly: error` with `all` (require braces everywhere)
- **Padding**: blank lines around `if`/`for`/`while`/`switch`/`try`/`throw`/`return`/`do`
- **Prettier**: 4-space indent, single quotes, `prettier-plugin-tailwindcss` for class sorting; ignores `ui/*` and mail views
- **Pint**: uses `laravel` preset
- **ESLint ignores** (not linted): `vendor`, `node_modules`, `public`, `bootstrap/ssr`, `vite.config.ts`, `resources/js/actions/**`, `resources/js/components/ui/*`, `resources/js/routes/**`, `resources/js/wayfinder/**`

## Testing

- Framework: **Pest 4** (not PHPUnit directly)
- Feature tests extend `Tests\TestCase` (bound in `tests/Pest.php`)
- **SQLite in-memory** DB in tests (no external DB needed)
- `RefreshDatabase` trait is available but **not** applied by default (commented out in `Pest.php`)
- CI runs tests against PHP 8.3, 8.4, 8.5

## Configuration & Environment

- `.npmrc` sets `ignore-scripts=true` — npm install will NOT run postinstall scripts
- `SESSION_DRIVER=database`, `CACHE_STORE=database`, `QUEUE_CONNECTION=database`
- `DB_CONNECTION=sqlite` by default; SQLite DB at `database/database.sqlite`
- `APP_KEY` must be generated via `php artisan key:generate` before use
- Wayfinder is configured with `formVariants: true` (generates form method variants for route helpers)
- React Compiler (babel-plugin-react-compiler) is enabled in Vite config
- Tailwind CSS v4 via `@tailwindcss/vite` plugin (no tailwind.config.js needed)

## Codegen (do NOT hand-edit)

The files under `resources/js/{actions,routes,wayfinder}/` are auto-generated by Laravel Wayfinder (`@laravel/vite-plugin-wayfinder`). They are regenerated when routes change. These are listed in `.prettierignore` and ESLint ignores. Do not modify them manually.
